## 📌 프로젝트 개요
Spring Boot 기반 간단한 Todo List API와 관리자 대시보드 기능을 제공하는 애플리케이션에 OpenTelemetry를 연동하여 Prometheus와 Grafana로 모니터링하는 시스템을 구축하는 프로젝트입니다. Docker 및 Kubernetes 환경에서 Helm Chart로 배포됩니다.

## 🛠 기술 스택
- Java 17
- Spring Boot 3.4.4
- Spring Data JPA
- DB: H2 (개발용) or MySQL
- OpenTelemetry SDK
- Prometheus
- Grafana
- Docker
- Kubernetes
- Helm
